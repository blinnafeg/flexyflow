# Вводный промпт для ИИ - FlexyFlow

Этот промпт предназначен для использования при работе с проектом FlexyFlow - визуальным конструктором страниц с системой действий и воркфлоу.

## Контекст проекта

FlexyFlow - это визуальный конструктор страниц с системой действий и воркфлоу, аналог FlutterFlow. Проект построен на следующем стеке:

- Vue 3 (Composition API, `<script setup>`)
- TypeScript (strict mode)
- Vite (сборщик)
- Tailwind CSS v4 (CSS-native, без config-файла)
- shadcn-vue (UI компоненты поверх reka-ui / radix-vue)
- Pinia (управление состоянием)
- Vue Router 4
- Supabase (self-hosted, БД и API)

## Основные сущности проекта

1. **Проекты** (Projects) - контейнеры для макетов, страниц, виджетов и воркфлоу
2. **Макеты** (Layouts) - сеточные структуры страниц с именованными слотами
3. **Страницы** (Pages) - конкретные страницы, использующие макеты и содержащие виджеты
4. **Виджеты** (Widgets) - переиспользуемые UI компоненты, созданные в визуальном редакторе
5. **Воркфлоу** (Workflows) - последовательности действий, реагирующие на триггеры
